A production-grade SaaS notification service — in-app, email, and webhook delivery via REST API with strict multi-tenant isolation.

> ⚡ **100 req/s** rate-limited per tenant · **26** integration tests passing

| Metric | Value |
|---|---|
| Async processing | BullMQ + Redis, 10 concurrent jobs, exponential backoff retry |
| Rate limiting | Nginx, 100 req/s per tenant API key |
| Real-time delivery | WebSockets, per-tenant rooms, JWT-authenticated |
| Deployment | Docker Compose, horizontally scaled (`--scale app=3`) |
| Test coverage | 26 integration tests (Jest + Supertest) |
